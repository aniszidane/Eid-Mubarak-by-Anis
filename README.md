# 🌙 Eid Mubarak by Anis

A premium Eid Al-Fitr greeting website built with **HTML**, **CSS**, and **JavaScript**.

This project was designed to give visitors a beautiful festive experience with:
- Personalized Eid greetings
- Elegant premium UI
- Golden night theme
- Countdown section
- Greeting cards
- Eid duas
- Animated effects
- Takbir audio on entry

## ✨ Features

- Premium Eid landing page design
- Personalized greeting with visitor name
- Smooth animated typing effect
- Welcome popup
- Royal animated visitor name
- Countdown to Eid / Eid prayer
- Copy and share greeting buttons
- WhatsApp sharing
- Save greeting as PNG
- Animated lanterns, moon, stars, and confetti
- Takbir audio support using `Taker.mp3`

## 📁 Project Structure

```bash
/project
  ├── index.html
  ├── style.css
  ├── script.js
  └── Taker.mp3
1. Clone the repository:
git clone https://github.com/your-username/Eid-Mubarak-by-Anis.git
2. Open the project folder:
cd Eid-Mubarak-by-Anis
3. Run the website by opening:
o index.html
Audio Note
The website uses a file named:
Taker.mp3
Make sure it is placed in the project root folder.
Some mobile browsers may block autoplay audio until the user interacts with the page once.
Customization
You can easily customize:
Project title in index.html
• Colors and premium styling in style.css
Greeting texts and countdown date in script.js
• Audio file by replacing Taker.mp3
